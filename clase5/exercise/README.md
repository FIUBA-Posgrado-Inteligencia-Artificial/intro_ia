# Evaluación práctica - Módulo 5 - Swiss Dataset

## 📋 Descripción del problema
En este ejercicio deberán implementar dos modelos de **regresión** para resolver un problema supervisado utilizando el conjunto de datos **Swiss**.

> El dataset **Swiss** es un conjunto de datos clásico que contiene información socioeconómica y demográfica de diferentes provincias de Suiza. Algunas de las variables incluidas son la tasa de fertilidad, el porcentaje de educación, la proporción de católicos, entre otras.

El objetivo consiste en construir modelos de regresión capaces de predecir la variable `Fertility` a partir del resto de las variables disponibles en el conjunto de datos.
