# Evaluación práctica - Módulo 6 - Detector de SPAM

## 📋 Descripción del problema
En este ejercicio deberán implementar dos modelos de **clasificación** para resolver un problema supervisado utilizando el conjunto de datos **Spambase**.

> El dataset **Spambase** contiene 4601 observaciones de correos electrónicos, de los cuales 2788 son legítimos y 1813 son SPAM. Los correos ya fueron preprocesados mediante técnicas de NLP: cada atributo representa la frecuencia normalizada de una palabra o símbolo en el mensaje.

El objetivo consiste en construir modelos de clasificación capaces de predecir si un correo es SPAM (`spam = 1`) o legítimo (`spam = 0`) a partir de las frecuencias de palabras disponibles en el dataset.
