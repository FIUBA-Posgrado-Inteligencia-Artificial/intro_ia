# Jupyter Notebooks - Tutoriales Prácticos

Este directorio contiene una serie de *notebooks* interactivas diseñadas como tutoriales prácticos y guiados para aprender a utilizar las herramientas fundamentales en Ciencia de Datos e Inteligencia Artificial.

Le sugerimos seguir el orden numérico de las notebooks, ya que las temáticas avanzan de forma progresiva.

## Listado de Notebooks:

### Introducción General
* **[1 - Primera Notebook.ipynb](1%20-%20Primera%20Notebook.ipynb)**: Primeros pasos en el entorno de Jupyter Notebook. Atajos, celdas de Markdown y celdas de código.
* **[2 - Usando Python.ipynb](2%20-%20Usando%20Python.ipynb)**: Repaso y ejercicios de conceptos básicos de la programación en Python.

### Manejo y Análisis de Datos (NumPy y Pandas)
* **[3 - Usando Numpy.ipynb](3%20-%20Usando%20Numpy.ipynb)**: Introducción a la computación científica eficiente con arreglos y matrices mediante la librería *NumPy*.
* **[4 - Usando Pandas - Introduccion.ipynb](4%20-%20Usando%20Pandas%20-%20Introduccion.ipynb)**: Primeros pasos explorando DataFrames y Series con la librería estrella para manipulación de datos tabulares, *Pandas*.
* **[5 - Usando Pandas - Operando.ipynb](5%20-%20Usando%20Pandas%20-%20Operando.ipynb)**: Operaciones avanzadas, filtros, selecciones, y transformaciones numéricas/condicionales sobre datos reales.
* **[6 - Usando Pandas - Combinando datos.ipynb](6%20-%20Usando%20Pandas%20-%20Combinando%20datos.ipynb)**: Técnicas de cruce de bases de datos: `concat`, `merge` y `join`.
* **[7 - Usando Pandas - Agrupando datos.ipynb](7%20-%20Usando%20Pandas%20-%20Agrupando%20datos.ipynb)**: La potencia operativa del paradigma "Split-Apply-Combine" usando la función `groupby`.
* **[8 - Usando Pandas - Fuentes de Datos.ipynb](8%20-%20Usando%20Pandas%20-%20Fuentes%20de%20Datos.ipynb)**: Cómo leer e importar desde internet o archivos locales conjuntos de datos en formatos tabulares (ej: `.csv`, Excel, o bases SQL).

### Visualización Gráfica
* **[9 - Usando Matplotlib.ipynb](9%20-%20Usando%20Matplotlib.ipynb)**: Creación de gráficos en 2D customizables aprovechando la librería base de diagramación de Python, *Matplotlib*.
* **[10 - Graficando con Pandas.ipynb](10%20-%20Graficando%20con%20Pandas.ipynb)**: Exploración rápida y visualización directa de la información aprovechando los métodos ploteables integrados *dentro* de todo DataFrame de Pandas.
* **[11 - Usando Seaborn.ipynb](11%20-%20Usando%20Seaborn.ipynb)**: Gráficos estadísticos avanzados y estéticamente refinados utilizando la librería *Seaborn*, la cual corre con un envoltorio de súper-alto nivel por encima de Matplotlib.

> **Nota sobre compatibilidad estructural:** La carpeta `datasets/` contiene todos los conjuntos de datos complementarios (`.csv` / tablas) necesarios que estos tutoriales consumen e importan localmente durante sus demostraciones de código.
