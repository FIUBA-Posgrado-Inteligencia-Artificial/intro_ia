# Ejercicios Prácticos de Fijación

Este directorio provee material complementario y no obligatorio para poner a prueba y afianzar los conocimientos adquiridos sobre las librerías típicas utilizadas en Ciencia de Datos.

Le sugerimos intentar resolver estos ejercicios por su cuenta luego de haber completado las [notebooks tutoriales](../jupyter_notebooks/README.md).

## Listado de Prácticas:

### Fundamentos
* **[1 - Ejercicios Python.ipynb](1%20-%20Ejercicios%20Python.ipynb)**: Consignas de programación general, bucles, funciones y lógica en Python puro.
* **[2 - Ejercicios Numpy.ipynb](2%20-%20Ejercicios%20Numpy.ipynb)**: Práctica intensiva de operaciones matriciales, *broadcasting* e indexación con NumPy.

### Manipulación y Análisis (Pandas)
* **[4 - Ejercicios Pandas - Homeless.ipynb](4%20-%20Ejercicios%20Pandas%20-%20Homeless.ipynb)**: Ejercicios iniciales de filtrado y manipulación usando un dataset sobre personas en situación de calle.
* **[5 - Ejercicios Pandas - Temperatura.ipynb](5%20-%20Ejercicios%20Pandas%20-%20Temperatura.ipynb)**: Prácticas de limpieza, conversión de tipos de datos e índices temporales.
* **[6 - Ejercicios Pandas - Aprender 2018.ipynb](6%20-%20Ejercicios%20Pandas%20-%20Aprender%202018.ipynb)**: Análisis sobre los resultados de las evaluaciones educativas Aprender.
* **[7 - Ejercicios Pandas - Estadística Descriptiva.ipynb](7%20-%20Ejercicios%20Pandas%20-%20Estadística%20Descriptiva.ipynb)**: Cálculo de métricas, tendencias centrales y dispersiones sobre datos tabulares.
* **[8 - Ejercicios Pandas - Agrupamiento.ipynb](8%20-%20Ejercicios%20Pandas%20-%20Agrupamiento.ipynb)**: Ejercitaciones avanzadas utilizando el método `groupby` y agregaciones complejas.

### Visualización
* **[3 - Ejercicios Matplotlib.ipynb](3%20-%20Ejercicios%20Matplotlib.ipynb)**: Retos de graficación, customización de ejes, títulos y múltiples *subplots* utilizando Matplotlib.
